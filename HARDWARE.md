# M1 Hardware Documentation

## MCU

- **Part:** STM32H573VIT6
- **Type:** 32-bit ARM Cortex-M33
- **Flash:** 2 MB
- **Package:** 100-pin LQFP

## Hardware Revision

- **Supported:** 2.x

## Development Tools

- STM32CubeIDE (recommended)
- J-Link or ST-Link debugger
- ARM GCC toolchain (14.2 or compatible)
